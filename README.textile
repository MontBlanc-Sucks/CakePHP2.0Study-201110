h1. 第2回CakePHP2.0勉強会@Tokyo

h3. 概要

CakePHPの次世代バージョン、CakePHP2.0の勉強会です。
6月に開催した "第1回":http://atnd.org/events/16634 に続いて2回目となります。
前回と違いより一般的なスタイルで、CakePHP2.0についての新機能などCakePHP2.0の話題を中心としたセッション、LT、事例紹介を行います。
例によって、お話しいただける方を募集します。詳細は下記をご覧ください。
